# Sprint_5
----
# Описание проекта 
В директории src лежат файлы :
конфиг с url и таймаутом
data со статичными данными авторизации
locators вынесены локаторы, используемые в тестовых сенариях
helpers генерация данных авторизации

Все тестовые сценарии лежать в директории tests

Инстанс драйвера поднимается и закрывается в фикстуре из conftest.py


# Автоматизированные тестовые сценарии
---- 
# 1. Кейс - Проверка регистрации нового пользователя<br>
test_registration_new_user.py<br>
1. Открываем главную страницу<br>
2. Нажимаем кнопку авторизации<br>
3. Нажимаем кнопку "Нет аккаунта"<br>
4. Заполняем форму регистрации<br>
5. Нажимаем кнопку "Создать аккаунт"<br>
6. Ждем перехода на главную страницу (появляется поле Поиск), появления имени юзера, аватарки<br>
7. Проверяем, что перешли на главную страницу (появилась поле поиска), виден аватар и имя юзера отображается корректно<br>
